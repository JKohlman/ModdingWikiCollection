CameraDroneUpgrades
Download
Mod Requirements
  • Qmodmanager
  • Smlhelper
  • placeholder
Mod Description
Another API for improving the camera drones. This mod by default also adds scanning functionality to all drones, without requiring an upgrade to be equipped. Any mod that uses this API will use an upgrade module that must be inserted into the scanner room upgrades slot in order for the drones to be able to use the upgrades.
